# 🚀 AppDelanto

Aplicación fintech para la gestión de adelantos de nómina de forma rápida, segura y eficiente.

---

## 📌 Descripción

**AppDelanto** permite a los empleados solicitar adelantos de su salario antes de la fecha de pago, con validación del empleador y gestión centralizada.

La plataforma está diseñada para facilitar el acceso a liquidez de manera controlada y confiable.

---

## 🎯 Objetivo

Brindar una solución digital que permita a los usuarios acceder a adelantos de nómina de forma sencilla, manteniendo control, seguridad y trazabilidad.

---

## ⚙️ Funcionalidades

* Registro de usuarios
* Solicitud de adelantos
* Validación de solicitudes
* Gestión de operaciones
* Historial de movimientos
